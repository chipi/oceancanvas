# Data Sources

All data used by OceanCanvas is free, open, and requires no registration or API key. The sources are government and intergovernmental scientific archives.

---

## Active sources

| Source | What it offers | Resolution | Freshness |
|---|---|---|---|
| [**NOAA OISST**](https://www.ncei.noaa.gov/products/optimum-interpolation-sst) | Sea surface temperature. Global, daily since 1981. Thermal colour fields, the Gulf Stream as art, anomaly maps. | 0.25° | 1–2 day |
| [**Argo Float Program**](https://argo.ucsd.edu) | ~4,000 autonomous floats measuring temperature and salinity from the surface to 2,000m. The observing network is itself visually compelling. | Float positions | ~10 days |
| [**GEBCO Bathymetry**](https://www.gebco.net) | Global ocean floor depth. Used as context data underlying surface renders. | 450m | Static |

---

## Future sources

| Source | What it offers |
|---|---|
| [**ESA SST-CCI**](https://climate.esa.int/en/odp/#/project/sst) | Gap-free SST at 0.05° resolution. Multi-decade, cross-calibrated. Better for long animations. |
| [**ESA Sea Level CCI**](https://climate.esa.int/en/odp/#/project/sea-level) | Sea level anomaly grids + global mean sea level time series. The 30-year rise curve. |
| [**ESA SSS-CCI**](https://climate.esa.int/en/odp/#/project/sea-surface-salinity) | Sea surface salinity. The Amazon freshwater plume. Evaporation patterns. |
| [**NSIDC Sea Ice**](https://nsidc.org/data/seaice_index) | Arctic/Antarctic ice concentration. The seasonal cycle. |
| [**ESA Ocean Colour CCI**](https://climate.esa.int/en/odp/#/project/ocean-colour) | Chlorophyll concentration. The spring phytoplankton bloom. |
| [**OSCAR Currents**](https://www.esr.org/research/oscar/) | Near-real-time surface current vectors. Natural for particle flow animations. |
| [**Open-Meteo Marine**](https://open-meteo.com/en/docs/marine-weather-api) | Wave height, wave period, wind speed. Audio drivers for the video editor. |
| [**NOAA NDBC Buoys**](https://www.ndbc.noaa.gov) | ~1,800 buoy stations. Waves, wind, SST, pressure. Point scatter renders. |
| [**SOCAT**](https://www.socat.info) | Ocean surface CO₂ measurements from ships. 1957 to present. |
| [**HadSST4**](https://www.metoffice.gov.uk/hadobs/hadsst4/) | UK Met Office SST anomaly record. 1850 to present. The long view. |

---

## Why these sources

Every source is chosen because it tells part of the ocean's story — and because it's genuinely interesting as visual material. SST shows the Gulf Stream. Argo shows the observing network. GEBCO shows the seafloor.

Each is free. Each is open. Each updates without human intervention. No API keys, no accounts, no institutional agreements.
