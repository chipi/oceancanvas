# Audio Attribution

## Ocean Theme

Synthetic ambient stems generated using ffmpeg audio synthesis (pink noise + sine tones). No third-party audio used.

- **License:** Public domain (generated procedurally)
- **Generation method:** ffmpeg `anoisesrc` (pink noise) + `sine` generators at varying intensity levels
- **5 stems:** calm → breathing → present → swelling → apex
- **Format:** WAV 44.1kHz 16-bit mono, 30 seconds each

## Future Themes
